# My Family Tree

Interactive and scalable family tree web application built with Next.js, TypeScript, Tailwind CSS, React Flow, and Supabase.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

To learn more about Next.js, take a look at the [Next.js Documentation](https://nextjs.org/docs).
